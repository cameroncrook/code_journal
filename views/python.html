Functions

Syntax

def {name}({parameters}):

def main():

if __name__ == "__main__":

main()

Python executes sequentually therfore, it will generate all the functions before the code within the functions are executed themselvs.

Placing all code in the `main` function allows you to code with functions and generates all functions before code is executed.

database=self.database, user=self.username,password=self.password, host=self.host, port=self.port

self.cursor = connection.cursor(cursor_factory = psycopg2.extras.RealDictCursor)
